== Unsupported Or Deferred Areas

Treat these as explicit decisions, not automatic rewrites:

* AF4 sagas.
* Upcasters/replay flows.
* Mongo event store/token store/DLQ.
* Kafka extension paths without AF5 support.
* Custom event-storage subclasses.
* SQL/DDL/data movement.
